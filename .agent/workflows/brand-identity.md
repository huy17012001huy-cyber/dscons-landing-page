# /brand-identity — Xây dựng Thương hiệu

## Nhiệm vụ
$ARGUMENTS

---

## DISCOVERY

```
1. Tên thương hiệu (đã có hoặc cần đặt)?
2. Ngành / lĩnh vực kinh doanh?
3. Target audience cụ thể?
4. 3 đối thủ cạnh tranh chính?
5. Điều gì làm bạn khác biệt?
6. Cảm xúc nào bạn muốn khách hàng cảm nhận?
7. Brand hiện tại có vấn đề gì không?
```

---

## AGENTS ĐIỀU PHỐI

### Bước 1 — Brand Strategy
**Agent:** `brand-design`
**Skills:** `brand-positioning-statement`, `brand-architecture`
- Competitive landscape analysis
- Positioning statement
- Brand personality và values
- Mission và vision statement

### Bước 2 — Naming (nếu cần)
**Agent:** `brand-design`
**Skills:** `naming-workshop`
- Naming criteria và guidelines
- 10+ name concepts
- Screening và shortlist
- Domain availability check brief

### Bước 3 — Brand Voice
**Agent:** `brand-design` + `content-writer`
**Skills:** `brand-voice-guide`, `brand-story`
- Tone of voice definition
- Writing do's and don'ts
- Brand story
- Tagline options

### Bước 4 — Visual Identity Brief
**Agent:** `brand-design`
**Skills:** `visual-identity-brief`, `color-palette-generator`
- Design direction (moodboard-ready brief)
- Color palette rationale
- Typography direction
- Logo brief cho designer

### Bước 5 — Messaging Framework
**Agent:** `content-writer`
**Skills:** `content-style-guide`
- Headline templates
- Value proposition variants
- Audience-specific messaging
- Content style guide

---

## DELIVERABLES

```
□ Brand Strategy Document (10-15 trang)
  - Positioning statement
  - Brand personality + values
  - Target audience portrait

□ Brand Identity Guidelines
  - Logo usage guidelines
  - Color palette (HEX, RGB, CMYK)
  - Typography system
  - Visual style direction

□ Brand Voice Guide
  - Tone + personality
  - Writing examples
  - Do's and don'ts

□ Messaging Framework
  - Tagline (3-5 options)
  - Elevator pitch
  - Core value propositions
```
