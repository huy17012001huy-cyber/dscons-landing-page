## Nhiệm vụ
$ARGUMENTS

---

## DISCOVERY

```
1. Sản phẩm/dịch vụ đang bán?
2. Giá bán (low-ticket <1M / mid-ticket 1-10M / high-ticket >10M VND)?
3. Target audience?
4. Traffic hiện tại (nếu có)?
5. Đã có email list chưa? Bao nhiêu subscribers?
6. Ngân sách cho paid traffic?
```

---

## FUNNEL TYPES THEO GIÁ

### Low-ticket Funnel (< 1M VND)
```
Opt-in page (Lead magnet)
    → Thank you + Tripwire offer
    → Upsell (core product)
    → Email sequence
Agent: email-automation + content-writer + ads-paid-media
```

### Mid-ticket Funnel (1-10M VND)
```
Content marketing / Ads
    → Lead magnet + Opt-in
    → Webinar / VSL
    → Sales page
    → Onboarding
Agent: sales-funnel + email-automation + events-speaking
```

### High-ticket Funnel (> 10M VND)
```
Content / Ads → Discovery call
    → Proposal
    → Contract
    → Onboarding
Agent: sales-funnel + email-automation + legal-compliance
```

## FUNNEL KPIs

```
Top of funnel:
- Cost per lead (CPL)
- Landing page conversion rate: >20-40% (opt-in)

Middle of funnel:
- Email open rate: >25%
- Click rate: >3%
- Webinar show rate: >30%

Bottom of funnel:
- Sales page conversion: 1-5%
- Cart abandonment rate: <70%
- Checkout conversion: >60%

Overall:
- ROAS: >3x (paid) / infinite (organic)
- Payback period: <30 days
```
