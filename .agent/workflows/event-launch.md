# /event-launch — Tổ chức Sự kiện

## Nhiệm vụ
$ARGUMENTS

---

## DISCOVERY

```
1. Loại sự kiện: webinar / workshop / conference / ra mắt / offline event?
2. Online hay offline?
3. Mục tiêu: leads / sales / brand / community?
4. Target audience và số lượng dự kiến?
5. Ngày tổ chức?
6. Budget?
7. Có bán hàng tại sự kiện không?
```

---

## AGENTS ĐIỀU PHỐI

### Bước 1 — Event Planning
**Agent:** `events-speaking`
**Skills:** `event-plan`, `webinar-plan`, `event-run-sheet`
- Agenda chi tiết
- Run sheet
- Speaker briefs
- Technical requirements

### Bước 2 — Content & Scripts
**Agent:** `events-speaking` + `content-writer`
**Skills:** `webinar-script`, `keynote-speech-structure`
- Opening hook (3 phút đầu)
- Main content script
- Q&A framework
- Closing CTA

### Bước 3 — Marketing
**Agent:** `social-media` + `content-writer`
**Skills:** `event-marketing-plan`, `event-promotion-content`
- Registration page copy
- Social media promotional content
- Countdown content (T-7, T-3, T-1, ngày sự kiện)

### Bước 4 — Email Campaign
**Agent:** `email-automation`
- Invitation email series
- Reminder sequence (T-7, T-1, T-3h)
- Post-event follow-up + replay

### Bước 5 — Post-event Follow-up
**Agent:** `sales-funnel` + `email-automation`
- 24h thank you + replay email
- Offer follow-up sequence
- Testimonial collection

---

## EVENT TIMELINE

```
T-4 tuần: Concept + registration page
T-3 tuần: Marketing content + emails
T-2 tuần: Ads live + social posts start
T-1 tuần: Reminder sequence begin
T-2 ngày: Final reminder + tech check
Event Day: Live execution
T+24h: Thank you + replay
T+3 ngày: Special offer deadline
T+7 ngày: Post-event debrief
```

---

## SUCCESS METRICS

```
Pre-event: Registration rate, Show-up rate (target >30-50%)
During: Engagement rate, Q&A participation
Post-event: Replay views, Conversion rate, NPS
```
