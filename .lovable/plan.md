

## DSCons Landing Page — Implementation Plan

A high-converting landing page for a BIM/Revit MEP training center, fully in Vietnamese, with a professional tech aesthetic.

### Design System
- **Font:** Inter (full Vietnamese diacritics support)
- **Colors:** Primary `#33AAEF`, Accent `#1978DC`, White/Light Gray backgrounds, Orange/Red CTAs for high contrast
- **Style:** Clean, professional, technical feel with generous whitespace

### Sections (top to bottom)

1. **Sticky Navbar** — Logo placeholder (left), brand name text, single CTA button "Nhận tư vấn ngay" (right, orange/red)

2. **Hero Section** — Pre-headline tag, bold H1 headline, descriptive sub-headline, video/image placeholder on the right, prominent "Đăng ký nhận ưu đãi Early Bird" CTA button

3. **Social Proof Bar** — Partner/software logo placeholders (Autodesk Revit, Navisworks) + 3 stats with icons (+1000 học viên, Hỗ trợ 24/7, Cam kết chất lượng)

4. **Benefits Grid** — 4 cards with icons: Kỹ năng thực chiến, Tài nguyên độc quyền, Hỗ trợ trọn đời, Chứng nhận & Sự nghiệp

5. **Course Roadmap** — Info box (schedule, tuition, instructor) + vertical timeline with 4 phases

6. **Testimonials** — Carousel/grid with student quotes, before/after project image placeholders

7. **FAQ Accordion** — 3 questions in Vietnamese with expandable answers (lorem ipsum)

8. **Bottom CTA** — Dark background, urgency messaging, large bright CTA button "Điền Form Đăng Ký Ngay"

9. **Footer** — Contact info, social icons (YouTube, Facebook, Zalo), legal links

### Components to Create
- `Navbar` — sticky, responsive
- `HeroSection` — split layout with text + media
- `SocialProof` — logo row + stats
- `BenefitsGrid` — 4 icon cards
- `CourseRoadmap` — info box + timeline steps
- `Testimonials` — carousel with quotes and image placeholders
- `FAQSection` — accordion using shadcn Accordion
- `BottomCTA` — full-width dark section
- `Footer` — contact, social, legal

All Vietnamese text preserved exactly as provided. Inter font loaded via Google Fonts.
